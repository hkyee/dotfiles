# Linux system setup for Herr Ken

This is a Repo for Herr Ken's portable Linux setup 


### 1.suckless
	- Autostart script called at /usr/share/xsessions/dwm.desktop
	- sudo make clean install

### 2.zsh
	- Put .zshrc in home
### 3.vim
	- Put .vimrc in home
	- Put .vim in home
### 4.nvim
	- Put folder in ~/.config
